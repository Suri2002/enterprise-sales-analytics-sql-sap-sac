# SAP Analytics Cloud Dashboards

This folder contains screenshots of dashboards built in SAP Analytics Cloud
using KPI views generated from SQL Server.

## Dashboards Included
- Executive Overview (Revenue, Profit, Margin)
- Revenue by Store
- Daily Revenue Trend

## Data Source
- SQL Server KPI Views
- Imported into SAC as models (CSV-based import)

These dashboards demonstrate enterprise-style BI reporting using SAP SAC.
